# Quadruped in OMPL

Implementation of the quadruped in OMPL. It has been treated as a unicycle. Primitives:
- rolling with 4 wheels (fw/bw/l/r)
- rolling with 2 wheels (fw/bw)
- turning on the spot
- stepping with one foot (fw/bw)


A gap is considered.
